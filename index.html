little shutdown program

Settings

Shutdown If enabled the computer will shut down when the timer ends
Close to system tray If enabled this program will minimize itself to the system tray instead of the taskbar
Native title bar If enabled a native titlebar will be displayed instead of the customized one
Auto update Search on start of the application for a new available version and notify if there is one available
Touch gestures Swipe left and right to get to the other screens
Background color Set the background color of the application
Text color Set the text color of the application
Title bar color Set the text/icon color of the title bar
